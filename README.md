# Monitoriamento-Parque-Serra-Mar


## Instalação
- cp .env.example .env
- composer install
- php artisan key:generate

## START

- php artisan serve
- Acesse 127.0.0.1:8000

## Banco de Dados

- https://serramar.tk/phpmyadmin
- DATABASE=bilac_serra
- USERNAME=bilac_serra
- PASSWORD=bilac2017
